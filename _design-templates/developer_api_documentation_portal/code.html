Authentication

The LuxeControl API uses API keys to authenticate requests. You can view and manage your API keys in the LuxeOps Dashboard.

Bearer Tokens

Authentication to the API is performed via HTTP Bearer Auth. Provide your API key as the bearer token value in the Authorization header.

info All API requests must be made over HTTPS. Calls made over plain HTTP will fail. API requests without authentication will also fail.

Manage API Keys

Use the panel below to generate or revoke access keys for your application. Keep your keys secure and do not share them in publicly accessible areas such as GitHub, client-side code, etc.
